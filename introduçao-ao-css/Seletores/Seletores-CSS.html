No CSS nós Possuimos Seletores de ID

  • Seletor de Tipo: Busca elementos por uma Tag HTML
  • Seletor de ID (#):Busca elementos através do atributo ID
  • Seletor de classe (.):Busca elementos através do atributo Classe
  • Seletor Universal (*):Selecionar todos os elementos HMTL
  • Seletor de Atributo([atrib]): Seleciona todos elementos que possuem um atributo especifico em nosso documento HTML e conseguimos tambem busca atributos com um valor especifico
