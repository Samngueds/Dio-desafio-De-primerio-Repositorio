Outra Forma de Declaração do CSS chamada "CSS Externo"

Outra forma de Declaração e o CSS Externo: e quando adicionamos todos os atributo Style na tag head no cabeçalho do HTML atravez de um arquivo separado aquivo .css que chamamos pra o arquivo principal pela tag link, que tambem vai servir pra formata e deixa seu HTML mais Bonito sem precisa altera o head da pagina e deixado mais automatico, antes de carrega pro cliente e em vez de ser de elemento por elemento coloca a regra em toda a pagina html e em outras pra algum espesificos.

Exemplo: ( <'head'> 'link ref="stylesheet" href="caminho do arquivo css" )

Assim linkando o arquivo com todas as configurações deixando mais facil de fazer a manutenção e alteração do css e o html
